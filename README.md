# Duck Clicker Game 💥

Ламповый 2D-кликер на базе библиотеки **Pyglet**, написанный в рамках практики объектно-ориентированного программирования и работы с игровыми фреймрейтами.

> **Примечание:** Мне 11 лет. Код написан мной самостоятельно при поддержке ИИ, без слепого копирования готовых листингов.

## 🛠 Технологический стек
* **Language:** Python 3.14
* **Framework:** Pyglet (работа с окнами, событиями мыши и планировщиком задач)
* **Architecture:** Event-driven (обработка событий ввода через декораторы `@window.event`)

## 🚀 Реализованные фичи
* **Delta Time Speed Independent:** Движение утки привязано к дельте времени (`dt`), что гарантирует одинаковую скорость игры на мониторах с любой герцовкой (60Hz / 144Hz) и защищает от просадок FPS.
* **Collision Detection:** Кастомный алгоритм просчета отскоков объекта от границ окна (рикошеты).
* **Random Spawn:** Автоматический пересчет координат цели при точном попадании курсором в хитбокс.

## 📦 Как запустить проект

1. Клонируй репозиторий:
   git clone https://github.com

2. Установи зависимости:
   pip install pyglet

3. Запусти скрипт:
   python duckgame.py
